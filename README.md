WeIdentity 区块链接入模块
=============================================================

整体介绍
--------

> WeIdentity-Blockchain是WeIdentity的区块链接入模块，为WeIdentity提供区块链相关功能，包括调用WeIdentity的智能合约和使用区块链的基础通信等功能，而且可以适配多种不同区块链，使得WeIdentity更加通用。
